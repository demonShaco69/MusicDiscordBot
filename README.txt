Что это такое:
Дискорд бот, с возможностью проигрывать аудио с youtube и некоторыми другими функциями

Описание функций можно узнать введя "help" в чате дискорда
Если ввести сообщение, содержащее:
    fumo,
    pepe,
    ascii
    - случится что-то интересное

Что необходимо для работы:
для работы нужна java(я использовал 13 версию)

ВНИМАНИЕ!
Для того, чтобы программы работала, понадобятся api-ключи:
tenor(https://developers.google.com/tenor/guides/quickstart)
и discord( https://discord.com/developers/docs/getting-started)

Введите их в соответствующие поля в файле data\secretData.py
